This deck provides word combinations in German and their translations in Russian.

How I created the deck

I asked ChatGPT-4o to generate word combinations for each word in a CSV.

The procedure was as follows:

  1. Input the prompt: Give word combinations for German words with Russian translation, keeping each part of speech in its respective form, in CSV format with "|" as a separator.
  2. Input ~200 lines from the CSV with words.
  3. Repeat for the next blocks of words.
  4. When the Chat outputs lines without providing word combinations, repeat the whole procedure.
  5. When all blocks are ready, copy all lines, paste them into deck.csv, shuffle them.
