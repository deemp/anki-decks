О колоде

Колода основана на книге "Русско-узбекский тематический словарь. 9000 слов", 2014. Автор книги - Таранов Андрей Михайлович.

От автора колоды: Низкий поклон создателям словаря за их огромную работу! Они показали, что узбекский язык современный, выразительный и красивый.

Поля карточек

  • index - Номер карточки в колоде.
  • word_ru - Слово (или фраза) на русском
  • word_uz - Перевод word_ru на узбекский
  • topic_index - Номер темы.
  • topic_ru - Название темы на русском языке.
  • chapter_index - Номер главы.
  • chapter_ru - Название главы на русском языке.
  • part_index - Номер части.
  • part_ru - Название части на русском языке.
  • word_sound_ru - Аудио для word_ru.

Поиск

Поля карточек можно использовать для поиска (Searching).

Полезно изучить раздел с регулярными выражениями (Regular expressions).

Примеры запросов:
  • "word_ru:re:^по" - слово на русском, начинающееся на "по" (регистр неважен).

Фильтрованные колоды

Чтобы учить конкретные слова, можно сделать фильтрованную колоду (Filtered Decks):

  • В Anki Desktop: Tools -> Create Filtered Deck
  • В AnkiDroid: 3 точки наверху справа -> Create filtered deck

При создании нужно будет написать запрос.

Аудио

Аудио было сгенерировано при помощи плагина HyperTTS. Я использовал бесплатные голоса Google Translate.

Хороший TTS для узбекского есть в Microsoft Translator (Android). Ждем бесплатный TTS для узбекского от Google.
