About this deck

This deck is based on the content of the site learnenglishkids.britishcouncil.org.

This deck contains English words sorted by topics and then lexicographically, Russian translations of these words and topics, audio of English and Russian words, and relevant images.

Fields

  • index - The card number in the deck. These numbers start from 0. Cards are sorted by this field.
  • word_en - An English word.
  • word_ru - A Russian translation of the word_en.
  • topic_en - The English topic to which the word_en belongs.
  • topic_ru - The topic to which the word_en belongs.
  • word_sound_en - Audio of word_en.
  • word_sound_ru - Audio of word_ru.
  • picture - HTML code for the image that illustrates word_en.

Update the deck

You can update the deck without downloading a new version from AnkiWeb.

  1. Copy the data from this file into a local file.
  2. Import the local file into the deck using the appropriate note type. In the Import options, set:
    • Existing notes: Update
    • Match scope: Notetype and deck
  3. Update the audio using the HyperTTS plugin for the desktop version of Anki (see 1, 2, 3). Tips:
    • Use free Google Translate voices to avoid regenerating the audio.
    • Create presets to simplify future updates.
    • Select all cards and update the audio. HyperTTS will generate audio only for fields that have changed.
